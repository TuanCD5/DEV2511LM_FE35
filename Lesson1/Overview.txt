Bài lab nộp thông qua classroom, nộp bằng link git.

Một số câu lệnh git:
    - Kiểm tra version: git --version
    - Khởi tạo repo mới: git init
    - Clone repo từ GitHub: git clone <URL>
    - Kiểm tra trạng thái file: git status
    - Add file vào staging: 
        git add <tên-file>
        git add .
    - Commit: git commit -m "message"
